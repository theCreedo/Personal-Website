Login Account - An Android HackTX Project

Description

Written in Java and XML. Incorporates Parse (Transferred to Mongodb). Engaged in app design and database storage.

Project Summary

Creates an app that displays a login screen and has access to create an account or log into a profile. Accounts are saved with emails, usernames, and passwords into Parse, and the profile displays the username and can upload a photo as a profile picture.

Snail Tale - An Interactive Platformer Project

Description

Created an interactive scrolling platform game within a team (4) over several months for the Texas Computer Educators Association's 0100 Elements game programming competition. Received 2nd place and written in Java.

Project Summary

A snail goes through Summer, Fall, Winter, and Spring fighting slimes, going through normal, spiky, and disappearing platforms, and overcoming tricky obstacles to collect gems in order to save the world from seasonal destruction.

What Are The Odds - An Android Homecoming Project

Description

Developed in Java and XML. Engaged in app design and sound implementation

Project Summary

Creates an app that poses a dare, and when given a number, uses that to randomly generate a number that the user tries not to guess, or else the user must do the dare. After the 2nd dare, it poses a dare to go to homecoming, and plays a song.
